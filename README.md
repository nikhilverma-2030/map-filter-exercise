# Student Search Application (Map & Filter)

A simple Student Search Application built using HTML, CSS, and JavaScript. This project displays student records in card format and allows users to search for students by name. The search functionality is implemented using JavaScript's filter() and startsWith() methods.

## Features

- Display all student records dynamically
- Search students by name
- Responsive card-based layout
- Fixed navigation bar and search section
- Clean and user-friendly interface
- Uses JavaScript Array Methods (map() and filter())

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

─ index.html
─ style.css
─ script.js
─ README.md

## How It Works

1. Student data is stored in a JavaScript array of objects.
2. All student records are displayed when the page loads.
3. Users can enter a student name in the search box.
4. Clicking the Search button filters and displays matching records.
